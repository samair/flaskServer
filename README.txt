Project Description
=========================

Project is a smiple vendor portfolio, where multiple user register themselves as vendors and list products they sell.

Features:
1. User login
2. Google login
3. Product list via html and json
4. Vendor list via html and json
5. User registration

HOW TO USE:
Run
python3 populate_db.py
to get sample users and vendor information.

Used :
- Bootstrap for css and js
- SQL lite
